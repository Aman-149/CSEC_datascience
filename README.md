# Python Foundations Assignment
## Overview

This repository contains my solutions for the Python Foundations Daily Assignment.
The goal of this project is to demonstrate understanding of core Python concepts such as:

Functions
Loops
Conditionals
Input/Output handling
Basic problem-solving and algorithms

All tasks from Level 1, Level 2, and Level 3 have been implemented.

## Level 1 Tasks
✅ Prime Number Checker
Checks whether a number is prime
Uses optimized approach (√n)
✅ Factorial Calculator
Computes factorial using iteration
Handles invalid input (negative numbers)
✅ Reverse String
Reverses a given string
Uses Python slicing for efficiency
## Level 2 Tasks
✅ Largest Number in a List
Finds the maximum value manually (without built-in max())
✅ Vowel Counter
Counts vowels in a string
Handles both uppercase and lowercase
✅ Fibonacci Sequence
Generates Fibonacci sequence up to n terms
Demonstrates loop-based algorithm
## Level 3 Tasks
✅ CLI Calculator
Supports:
Addition (+)
Subtraction (-)
Multiplication (*)
Division (/)
Handles invalid operations and division by zero
Runs continuously until user exits
# How to Run
Make sure you have Python installed, then run any file:
python3 level1/prime.py
python3 level2/fibonacci.py
python3 level3/calculator.py
